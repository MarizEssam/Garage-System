MyGarage

Garage is a software managment sustem used to organize and coordinate movements inside any parking lot.
It can used by any business organization that has a parking lot. It has a simple and user-friendly interface to
be accessable for any user from any background.


Installation

Extract the .zip file
Run the .exe file
The software is ready to work.


Usage

All you have to do is check in via the "Car Enter button", then you will be asked about your status whether
Handicapped or not. Depending on your choice a list of the available parking spots will be shown in a form of
a map. Next, You will choose any spot you want to park into. A confirmation message will appear. 
For Check out press the "Car Leave" button. Choose your name and a detailed payment info will be generated for your
parking session.
